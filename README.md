# Turtle-Enemies

Un joc 2D construit cu Pygame, dezvoltat ca proiect practic in cadrul cursului "Python 101" la Cisco Academy.



## Functionalitati

- Miscare a personajului controlat de jucator  
- Generare si miscare a inamicilor  
- Detectarea coliziunilor intre jucator si inamici  
- Scor simplu sau sistem de vieti (cat timp supravietuiesti)  
- Elemente grafice si sonore (imagini, text, muzica) â€“ fisierele de resurse sunt incluse  

## Instalare

1. Asigura-te ca ai instalat Python (versiunea 3.x) pe sistemul tau  
2. Instaleaza Pygame, daca nu e deja instalat:  
   ```bash
   pip install pygame
3. Cloneaza acest repository:
   ```bash
   git clone https://github.com/LSStefan/Turtle-Enemies.git
4. Acceseaza directorul proiectului:
   ```bash
   cd Turtle-Enemies
5. Ruleza jocul:
   ```bash
   python3 main.py
