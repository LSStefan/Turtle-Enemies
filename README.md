Turtle-Enemies

A simple 2D game built with Pygame, developed as part of the Python 101 course at Cisco Academy.
Turtle-Enemies is a small 2D arcade-style game created during the Python 101 course at Cisco Academy, as a practical project to apply the fundamentals of Python programming and game development with Pygame.
The project focuses on understanding the game loop, sprite management, and enemy interactions, serving as a foundation for beginners who want to explore how games are structured in Python.


Developed by Stefanita Lican during the Python 101 course at Cisco Academy.
